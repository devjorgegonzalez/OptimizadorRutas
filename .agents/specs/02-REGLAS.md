# Reglas para la solucion

## Para la IA

- Cada vez que se te pida generar un nuevo modulo, deberas repasar los archivos
  1. 00-CONTEXTO.md
  2. 01-ARQUITECTURA.md

## Generales

- Todos los metodos, modelos, componentes, rutas, etc. Deben ser en castellano.
- Todo el codigo debe estar documentado.
- Todo el codigo debe seguir la reglas de Domain Driven Design.
- Revisaras la carpeta skills.

## Front

- Debe ser responsivo.
- Debe tener modo claro y oscuro.
- Usaras solo Client Side Rendering.
- Las claves se manejaran en variables de entorno.

## Back

- Usar inyeccion de dependencias.
- Todos los endpoints y clases de entrada de los endpoint deberan tener los decoradores swagger necesarios para la documentacion.
